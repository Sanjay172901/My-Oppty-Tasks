11. Text Formatting

Styling text with CSS

CSS Text Properties

CSS provides numerous properties to format and style text content. These properties control the appearance, alignment, spacing, and decoration of text.

Text Color

The color property sets the color of text:

.red-text { color: red; }
.hex-text { color: #ff6b6b; }
.rgb-text { color: rgb(255, 107, 107); }
.rgba-text { color: rgba(255, 107, 107, 0.8); }

Text Alignment

The text-align property controls the horizontal alignment of text:

.left-align { text-align: left; }
.center-align { text-align: center; }
.right-align { text-align: right; }
.justify-align { text-align: justify; }

Text Decoration

The text-decoration property adds decorative lines to text:

.underline { text-decoration: underline; }
.overline { text-decoration: overline; }
.line-through { text-decoration: line-through; }
.no-decoration { text-decoration: none; }

Text Transformation

The text-transform property controls capitalization:

.uppercase { text-transform: uppercase; }
.lowercase { text-transform: lowercase; }
.capitalize { text-transform: capitalize; }
.none-transform { text-transform: none; }

Text Spacing

Letter Spacing

.normal-spacing { letter-spacing: normal; }
.wide-spacing { letter-spacing: 3px; }
.tight-spacing { letter-spacing: -1px; }

Word Spacing

.normal-word-spacing { word-spacing: normal; }
.wide-word-spacing { word-spacing: 10px; }
.tight-word-spacing { word-spacing: -2px; }

Line Height

.normal-line-height { line-height: normal; }
.spaced-line-height { line-height: 1.8; }
.tight-line-height { line-height: 1.2; }

Text Shadow

.simple-shadow { text-shadow: 2px 2px 3px #333; }
.multiple-shadows {
  text-shadow:
  1px 1px 2px #ff6b6b,
  0 0 1em #ee5a24,
  0 0 0.2em #667eea;
}
.no-shadow { text-shadow: none; }

Text Overflow

Control how text behaves when it overflows:

.clip-text {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: clip;
}

.ellipsis-text {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}
