10. Box Model

Understanding the CSS box model

What is the CSS Box Model?

The CSS box model is essentially a box that wraps around every HTML element. It consists of: content, padding, borders, and margins. Understanding the box model is crucial for CSS layout and sizing. 

/* Box Model Visualization */
.content { width: 200px; height: 100px; background-color: #ff6b6b; }
.padding { padding: 20px; background-color: #4ecdc4; }
.border { border: 5px solid #45b7d1; }
.margin { margin: 30px; }

Box Model Components

1. Content

The content area is where your text and images appear. The size of this area is determined by width and height. 

.content-box {
  width: 200px;
  height: 100px;
  background-color: #ff6b6b;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

2. Padding

Padding is the space between the content and the border. It is transparent and takes the background color of the element. 

.padding-box {
  width: 200px;
  height: 100px;
  padding: 20px;
  background-color: #4ecdc4;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

3. Border

The border surrounds the padding and content. It can have different styles, widths, and colors. 

.border-box {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid #45b7d1;
  background-color: #4ecdc4;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

4. Margin

Margin is the space outside the border. It separates the element from other elements. 

.margin-box {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid #45b7d1;
  margin: 30px;
  background-color: #4ecdc4;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

Box-sizing Property

The box-sizing property controls how total width and height are calculated.

content-box (Default)

.content-box-sizing {
  box-sizing: content-box;
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid;
  /* Total width = 200 + 40 + 10 = 250px */
}

border-box

.border-box-sizing {
  box-sizing: border-box;
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid;
  /* Padding and border included inside 200px */
}

Calculating Total Width and Height

With content-box (default)

Total width = width + left padding + right padding + left border + right border + left margin + right margin 

.example-content-box {
  width: 200px;
  padding: 10px 20px;
  border: 5px solid;
  margin: 15px 25px;
  /* Total width = 200 + 20 + 20 + 5 + 5 + 25 + 25 = 300px */
}

With border-box

Total width = width (includes padding & border) + left margin + right margin 

.example-border-box {
  box-sizing: border-box;
  width: 200px;
  padding: 10px 20px;
  border: 5px solid;
  margin: 15px 25px;
  /* Total width = 200 + 25 + 25 = 250px */
}
