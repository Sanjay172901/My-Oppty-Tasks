7. CSS Margins

Learn how to control space outside elements.

Basic Margin Examples

.box1 { margin: 20px; }
.box2 { margin: 10px 40px; }
.box3 { margin-top: 50px; }
.box4 { margin: -10px; }

CSS Margin Properties

The CSS margin properties are used to create space around elements, outside of any defined borders. Margins can be set for all sides of an element or for specific sides.

Margin Values

Margin values can be specified in several ways:

/* Length values */
.length-margin { margin: 20px; }

/* Percentage values */
.percentage-margin { margin: 5%; }

/* Auto value (for centering) */
.auto-margin { margin: 0 auto; } /* Centers block element */

/* Initial and inherit */
.initial-margin { margin: initial; }
.inherit-margin { margin: inherit; }

Individual Margin Sides

You can set margins for specific sides of an element:

.top-margin { margin-top: 20px; }
.right-margin { margin-right: 15px; }
.bottom-margin { margin-bottom: 30px; }
.left-margin { margin-left: 10px; }

Margin Shorthand

The margin property is a shorthand for setting all margin properties in one declaration: 

/* All sides same */
.all-sides { margin: 20px; }

/* Vertical and horizontal */
.vertical-horizontal { margin: 10px 20px; } /* top/bottom left/right */

/* Top, horizontal, bottom */
.top-horizontal-bottom { margin: 10px 20px 15px; }

/* Top Right Bottom Left */
.all-four { margin: 10px 20px 15px 25px; }

Margin Collapsing

Top and bottom margins of adjacent elements can collapse (combine) into a single margin: 

.element1 { margin-bottom: 30px; }
.element2 { margin-top: 20px; }

/* The actual margin between them will be 30px (the larger value) */

Negative Margins

Negative margins can be used to overlap elements or pull them outside their container: 

.negative-margin {
  margin-top: -20px;
  margin-left: -10px;
}

Auto Margins

Auto margins are commonly used to center block elements horizontally: 

.centered {
  width: 300px;
  margin: 0 auto; /* Centers horizontally */
}

.left-aligned {
  width: 300px;
  margin: 0;
}

.right-aligned {
  width: 300px;
  margin: 0 0 0 auto; /* Right alignment */
}

Margin with Different Units

Margins can be specified using different CSS units:

.pixel-margin { margin: 20px; }

.em-margin { margin: 2em; } /* Relative to font size */

.rem-margin { margin: 1.5rem; } /* Relative to root font size */

.percent-margin { margin: 5%; } /* Relative to parent element */

.viewport-margin { margin: 2vw; } /* Relative to viewport width */
