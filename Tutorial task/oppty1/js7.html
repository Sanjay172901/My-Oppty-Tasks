7. JavaScript Arrays

Store multiple values in a single variable using ordered lists

What are Arrays?

An array is a special variable, which can hold more than one value at a time. If you have a list of items (a list of car names, for example), storing the cars in single variables could look like this:

let car1 = "Saab";
let car2 = "Volvo";
let car3 = "BMW";

However, what if you want to loop through the cars and find a specific one? And what if you had 300 cars, not 3? The solution is an array!

Creating an Array

Using an array literal is the easiest way to create a JavaScript Array.

const cars = ["Saab", "Volvo", "BMW"];

You can also create an array, and then provide the elements:
const cars = []; cars[0]= "Saab"; cars[1]= "Volvo";

Accessing Elements

You access an array element by referring to the index number. JavaScript array indexes start with 0.

const cars = ["Saab", "Volvo", "BMW"];
let car = cars[0]; // Accesses "Saab"

// Changing an element
cars[0] = "Opel"; // Now cars is ["Opel", "Volvo", "BMW"]

Common Array Methods

JavaScript provides powerful methods to manipulate arrays:

Adding/Removing

  • push(): Add to end
  • pop(): Remove from end
  • shift(): Remove from start
  • unshift(): Add to start

Transforming

  • sort(): Sort alphabetically
  • reverse(): Invert order
  • join(): Convert to string
  • concat(): Merge arrays

Array Length Property

The length property of an array returns the number of elements in it. It is always one higher than the highest array index.

const fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.length; // Returns 4
