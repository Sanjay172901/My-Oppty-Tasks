14.CSS Float & Modern Layouts

Creating Layouts with Float

The float property specifies whether an element should float to the left, right, or not at all.

Float Left

Text wraps around the element floated to the left.

Float Right

Text wraps around the element floated to the right.

Float None

This element stays in normal document flow.

Clear Property

Clear Left

.clear-left { clear: left; }

Clear Right

.clear-right { clear: right; }

Clear Both

.clear-both { clear: both; }

Clear None

.clear-none { clear: none; }

Float vs Modern Layouts

Float (Legacy)

.float-layout .sidebar { float:left; width:25%; }
.float-layout .main { float:right; width:70%; }

Flexbox (Modern)

.flex-layout { display:flex; }
.flex-layout .sidebar { flex:1; }
.flex-layout .main { flex:3; }

Grid (Modern)

.grid-layout {
display:grid;
grid-template-columns:1fr 3fr;
}

Float requires clearing and manual width control.
Flexbox is great for one-dimensional layouts.
Grid is best for two-dimensional layouts.
