1. Introduction to JavaScript

Learn what JavaScript is and how it powers the web

What is JavaScript?

JavaScript is a versatile programming language that adds interactivity to websites. It's one of the core technologies of the web, alongside HTML and CSS.

Originally created by Brendan Eich in 1995, JavaScript has evolved into a powerful language that runs not only in browsers but also on servers and mobile devices.

The JavaScript Engine

JavaScript doesn't just "run"; it is interpreted and compiled by an engine. The most famous is Google's V8 Engine (used in Chrome and Node.js).

Why Learn JavaScript?

  • Ubiquity: JavaScript runs in every major web browser.
  • Versatility: Frontend, backend, mobile, and desktop applications.
  • Large Community: Extensive ecosystem of libraries like React and Vue.
  • High Demand: One of the most sought-after skills in tech.

How JavaScript Works

JavaScript executes in the browser and can manipulate HTML and CSS to create dynamic, interactive web pages. It responds to user actions like clicks, form submissions, and keyboard input.

Client-side vs Server-side

Initially strictly Client-side, JavaScript can now run on the Server-side via Node.js, enabling full-stack application development with one language.
