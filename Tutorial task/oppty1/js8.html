8. Array Iteration

Master powerful methods to loop through and transform array data

What is Array Iteration?

Array iteration methods operate on every array item. They allow you to perform actions, transform data, or filter elements without using traditional for loops.

The forEach() Method

The forEach() method calls a function (a callback function) once for each array element.

const numbers = [45, 4, 9, 16, 25];
numbers.forEach(myFunction);

function myFunction(value, index, array) {
  console.log(value);
}

Note: forEach() does not create a new array. Use it for side effects (like logging or updating the DOM).

The map() Method

The map() method creates a new array by performing a function on each array element. It does not change the original array.

const numbers1 = [45, 4, 9, 16, 25];
const numbers2 = numbers1.map(x => x * 2);

// numbers2 is now [90, 8, 18, 32, 50]

The filter() Method

The filter() method creates a new array with array elements that pass a test.

const numbers = [45, 4, 9, 16, 25];
const over18 = numbers.filter(value => value > 18);

// over18 is now [45, 25]

The reduce() Method

The reduce() method runs a function on each array element to produce (reduce it to) a single value. It works from left-to-right.

const numbers = [45, 4, 9, 16, 25];
let sum = numbers.reduce((total, value) => total + value);

// sum is now 99

You can also provide an initial value for the total: numbers.reduce(func, 100);

Additional Iteration Methods

every() & some()

Check if all (every) or some (some) elements pass a test.

  • Returns Boolean

find() & findIndex()

Returns the value or the index of the first element that passes a test.
