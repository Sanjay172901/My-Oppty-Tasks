8. Images in HTML

Images are added in HTML using the <img> tag. Images improve visual appearance, enhance user engagement, and help explain content more effectively.

Basic Image Syntax

The <img> tag is a self-closing tag. It requires the src attribute to define the image path and the alt attribute for alternative text. 

<img src="https://via.placeholder.com/200" alt="Sample Image">

Output

Understanding Image Attributes

The <img> tag supports several important attributes:

  • src – Specifies the image file location.
  • alt – Describes the image for accessibility and SEO.
  • width – Sets image width.
  • height – Sets image height.
  • title – Tooltip text shown on hover.

Setting Width and Height

You can control image size using width and height attributes. 

<img src="https://via.placeholder.com/300" 
width="150" height="150" 
alt="Resized Image">

Output

Relative vs Absolute Paths

Images can be loaded using:

  • Absolute Path – Full URL (https://example.com/image.jpg)
  • Relative Path – Local file path (images/photo.jpg)
<img src="images/photo.jpg" alt="Local Image">

Output

Image as a Link

Images can also be clickable by wrapping them inside an anchor tag. 

<a href="https://www.google.com">
  <img src="https://via.placeholder.com/150" alt="Clickable Image">
</a>

Output

Why alt Attribute is Important

The alt attribute:

  • Improves accessibility for screen readers
  • Displays text if image fails to load
  • Helps search engines understand the image

Always write meaningful alt text describing the image content.
