3. HTML Attributes

HTML attributes provide additional information about HTML elements. They are always written in the start tag and usually appear as name="value".

Basic Example

<a href="https://example.com">Visit Example</a>
<img src="image.jpg" alt="Description" width="500" height="600">

1. href Attribute

Specifies the URL of the page the link goes to.

<a href="https://www.google.com">Google</a>

2. src Attribute

Specifies the path of an image.

<img src="mountain.jpg">

3. alt Attribute

Provides alternate text if the image cannot load.

<img src="mountain.jpg" alt="Mountain View">

4. width and height

Defines the size of an element.

<img src="mountain.jpg" width="300" height="200">

5. style Attribute

Adds inline CSS styling.

<p style="color:blue; font-size:20px;">
This is blue text
</p>

6. class Attribute

Used to group elements for styling.

<div class="box">Content</div>

7. id Attribute

Gives a unique identification to an element.

<p id="intro">Introduction paragraph</p>

8. title Attribute

Shows extra information when you hover over the element.

<p title="Tooltip text">Hover over me</p>

Basic Example

<!DOCTYPE html>
<html>
<body>

<h2 style="color:blue; text-align:center;" title="This is a heading">
Welcome to HTML Attributes
</h2>

<a href="https://www.google.com"
   target="_blank"
   title="Go to Google">
Visit Google
</a>

<br><br>

<img src="https://picsum.photos/150" 
     alt="Random Image" 
     width="150" 
     height="150" 
     style="border-radius:10px; border:3px solid black;">

<p id="demo" class="text"
   style="color:green; font-size:18px;">
This paragraph uses id, class and style attributes.
</p>

</body>
</html>

Output

Additional Notes

• Attributes are always written inside the opening tag.
• Attribute values should be enclosed in double quotes.
• Some attributes are global and can be used on any element.
• Attributes improve accessibility and functionality of webpages.
