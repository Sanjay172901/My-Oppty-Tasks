2. HTML Elements and Tags

What are HTML Elements?

An HTML element is everything from the start tag to the end tag. 

<p>This is a paragraph.</p>
<h1>This is a heading.</h1>
<br>

• <p>This is a paragraph.</p> is an HTML element
• <h1>This is a heading.</h1> is another HTML element
• <br> is an empty element (no closing tag)

HTML Tags

HTML tags are keywords surrounded by angle brackets. 

<tagname>Content goes here...</tagname>

• Tags normally come in pairs (start and end tag).
• The end tag contains a forward slash (/).
• Some tags are empty like <br>.

Common HTML Elements

<h1>Main heading</h1>
<h2>Subheading</h2>
<p>This is a paragraph.</p>
<a href="https://example.com">This is a link</a>
<img src="image.jpg" alt="Description">
<div>A container element</div>
<span>Inline container</span>
