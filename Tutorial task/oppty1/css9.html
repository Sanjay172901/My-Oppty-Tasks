9. Height & Width

Setting dimensions of elements

CSS Height and Width Properties

The CSS height and width properties are used to set the height and width of an element. These properties do not include padding, borders, or margins.

Height Property

The height property sets the height of an element:

.auto-height { height: auto; } 
.fixed-height { height: 200px; }
.percentage-height { height: 50%; }
.max-height { max-height: 300px; }
.min-height { min-height: 100px; }

Width Property

The width property sets the width of an element:

.auto-width { width: auto; }
.fixed-width { width: 300px; }
.percentage-width { width: 75%; }
.max-width { max-width: 500px; }
.min-width { min-width: 200px; }

Common Units

Height and width can be specified using various CSS units:

.pixel-dimensions {
  width: 200px;
  height: 150px;
}

.em-dimensions {
  width: 20em;
  height: 15em;
}

.rem-dimensions {
  width: 15rem;
  height: 10rem;
}

.percent-dimensions {
  width: 80%;
  height: 60%;
}

.viewport-dimensions {
  width: 50vw;
  height: 30vh;
}

.viewport-min {
  width: 50vmin;
  height: 30vmin;
}

.viewport-max {
  width: 30vmax;
  height: 20vmax;
}

Max and Min Dimensions

These properties set constraints on element dimensions:

.max-constrained {
  width: 100%;
  max-width: 800px;
  height: 200px;
  max-height: 300px;
}

.min-constrained {
  width: 200px;
  min-width: 150px;
  height: 100px;
  min-height: 80px;
}

.responsive-box {
  width: 100%;
  max-width: 600px;
  min-width: 300px;
  height: 200px;
  min-height: 150px;
}

Box-sizing Property

The box-sizing property affects how width and height are calculated:

.content-box {
  box-sizing: content-box;
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid #ff6b6b;
}

.border-box {
  box-sizing: border-box;
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid #667eea;
}
