1. Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language used to create webpages. HTML describes the structure of a webpage using elements.

What is HTML?

HTML is not a programming language. It is a markup language used to structure content on the web. It tells the browser how to display text, images, links, videos, and other content.

History of HTML

HTML was created by Tim Berners-Lee in 1991. It was developed to share documents over the internet. Since then, HTML has evolved through multiple versions. The latest version is HTML5.

Basic Structure of an HTML Document

Every HTML document follows a basic structure that includes: Document type declaration, head section, and body section. 

<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>

Output

Main Parts of HTML Page

  • <!DOCTYPE> – Defines the document type
  • <html> – Root element of the page
  • <head> – Contains metadata
  • <title> – Title shown in browser tab
  • <body> – Contains visible content

The <head> section contains metadata like title and styles, while the <body> section contains visible content shown to users.

HTML Versions

HTML has gone through many versions since its creation. Each version introduced new features and improvements.

  • HTML 1.0 (1991) – Basic document structure
  • HTML 2.0 (1995) – Standardized version
  • HTML 4.01 (1999) – Added CSS support
  • XHTML (2000) – Stricter version of HTML
  • HTML5 (2014) – Added audio, video, canvas, semantic elements

HTML5 is the latest and most widely used version today. It provides modern features required for web applications.

Features of HTML

  • Easy to learn and use
  • Platform independent
  • Supports multimedia (images, videos, audio)
  • Works with CSS and JavaScript
  • Supported by all browsers

Why Learn HTML?

HTML is the foundation of web development. To become a Frontend or Full Stack Developer, HTML is the first and most important step. It helps you understand how webpages are structured.

Simple Example

<h1>Welcome to My Website</h1>
<p>This is my first webpage.</p>
<a href="https://www.google.com">Visit Google</a>

Output

This example shows a heading, paragraph, and a link. These are some of the most commonly used HTML elements.
