13.Display

Display Property

Block Elements

Block elements start on a new line and take up the full width available:

.block-element { display: block; width: 300px; height: 50px; background-color: #ff6b6b; margin: 10px 0;

Inline Elements

Inline elements flow within text and only take up as much width as necessary: 

.inline-element { display: inline; width: 300px; height: 50px; background-color: #ff6b6b; margin: 10px; }

Inline-Block Elements

inline-block-element { display: inline-block; width: 100px; height: 50px; background-color: #667eea; margin: 10px; }

None

The none value completely removes the element from the document flow 

.hidden-element { display: none; }

Flex

Flex display creates a flexible box layout 

    .flex-container { display: flex; gap: 10px; }
     .flex-item { flex: 1; padding: 20px; background-color: #ff6b6b; }

Grid

Grid display creates a grid layout 

grid-container { display: grid; grid-template-columns: repeat(3, 1fr); gap: 10px; } 
.grid-item { padding: 20px; background-color: #4ecdc4; }

Visibility Property

Visible

The visible value makes an element visible (default) 

/* .visible-element { visibility: visible; }

Hidden

The hidden value hides an element but keeps it in the document flow

.hidden-element { visibility: hidden; }

Display vs Visibility

Understanding the key differences between display: none and visibility: hidden

/* display: none removes the element from the document flow */ 
.display-none { display: none; } 
/* visibility: hidden hides the element but keeps its space */ .visibility-hidden { visibility: hidden; }
