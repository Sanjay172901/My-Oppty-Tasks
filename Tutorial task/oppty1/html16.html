16. HTML Iframe Tag

The <iframe> tag is used to embed another webpage inside the current webpage. It creates an inline frame.

1. Basic Iframe

Embeds another website inside your page.

<iframe src="https://example.com"></iframe>

2. Width and Height

Defines iframe size.

<iframe src="https://example.com" 
width="400" height="200"></iframe>

3. Remove Border

Use CSS to remove iframe border.

<iframe src="https://example.com" 
style="border:none;"></iframe>

4. name Attribute

Used to target links.

<iframe name="myFrame"></iframe>

<a href="https://example.com" 
target="myFrame">Open in Iframe</a>

5. sandbox Attribute

Adds security restrictions.

<iframe src="https://example.com" 
sandbox></iframe>

6. loading Attribute

Lazy loads iframe.

<iframe src="https://example.com" 
loading="lazy"></iframe>

7. allow Attribute

Controls special permissions like fullscreen.

<iframe src="https://example.com" 
allow="fullscreen"></iframe>

8. referrerpolicy Attribute

Controls referrer information sent.

<iframe src="https://example.com" 
referrerpolicy="no-referrer"></iframe>
