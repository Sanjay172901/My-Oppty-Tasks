2. CSS Syntax

Understanding CSS selectors, properties, and values.

CSS Rule Structure

A CSS rule consists of a selector and a declaration block. The declaration block contains one or more declarations separated by semicolons. Each declaration includes a CSS property name and a value, separated by a colon. 

selector {
    property: value;
    property: value;
}

CSS Selectors

Selectors are patterns used to select the HTML elements you want to style.

Element Selector

Selects elements based on their tag name. 

h1 { color: blue; }
p { font-size: 16px; }

ID Selector

Selects an element based on its unique id attribute. Use the # symbol before the id name. 

#header {
    background-color: #333;
    color: white;
}

Class Selector

Selects elements based on their class attribute. Use the . symbol before the class name. 

.highlight {
    background-color: yellow;
}

Universal Selector

Selects all elements on the page using the * symbol. 

* {
    margin: 0;
    padding: 0;
}

Grouping Selectors

You can group multiple selectors using commas to apply the same styles. 

h1, h2, h3 {
    color: #ff6a00;
}

CSS Properties and Values

Properties define what aspect of an element you want to style, and values define what you want to set that property to.

Color Properties

CSS supports different ways to define colors: named colors, hex codes, RGB, and RGBA values. 

.color-example {
    color: red;          /* Named color */
    color: #ff0000;      /* Hex code */
    color: rgb(255,0,0); /* RGB values */
    color: rgba(255,0,0,0.5); /* RGBA with opacity */
}

Size Properties

CSS allows you to define sizes using different units like pixels, percentages, em, and rem. 

.size-example {
    width: 100px;   /* Pixels */
    height: 50%;    /* Percentage */
    margin: 1em;    /* Relative to font size */
    padding: 2rem;  /* Relative to root font size */
}

Shorthand Properties

Some CSS properties can be written in shorthand to save time and reduce code length. 

/* Longhand */
.margin-example {
    margin-top: 10px;
    margin-right: 20px;
    margin-bottom: 10px;
    margin-left: 20px;
}

/* Shorthand */
.margin-example {
    margin: 10px 20px;
}

/* Padding shorthand */
.padding-example {
    padding: 10px 20px 15px 25px;
}
