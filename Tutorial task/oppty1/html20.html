20. HTML Entities & Responsive Design

HTML Entities allow us to display reserved and special characters in HTML correctly. Responsive Web Design ensures that websites adapt smoothly to different screen sizes such as mobile phones, tablets, and desktops.

Part 1: HTML Entities

Some characters in HTML are reserved because they are used in tag syntax. If used directly, the browser interprets them as HTML code. To display them as text, we use HTML Entities.

Reserved Characters

  • < (less than)
  • > (greater than)
  • & (ampersand)
  • " (quotation mark)

Entity Syntax

Entities begin with & and end with ; 

<p>5 &lt; 10</p>
<p>10 &gt; 5</p>
<p>Tom &amp; Jerry</p>

Common HTML Entities

&lt;   → <
&gt;   → >
&amp;  → &
&quot; → "
&nbsp; → Non-breaking space
&copy; → ©
&reg;  → ®

Non-Breaking Space

HTML normally collapses multiple spaces into one. Use &nbsp; to create extra space without breaking line. 

<p>Hello&nbsp;&nbsp;&nbsp;World</p>

Numeric Entities

&#169;  → ©
&#174;  → ®
&#8364; → €

Numeric entities are useful when named entities are unavailable.

Part 2: Responsive Web Design

Responsive Web Design (RWD) makes web pages look good on all devices. It uses flexible layouts, images, and CSS media queries.

1. Viewport Meta Tag

<meta name="viewport" content="width=device-width, initial-scale=1.0">

This tells the browser how to scale the page on mobile devices.

2. Flexible Layout

.container {
  width: 90%;
  margin: auto;
}

3. Responsive Images

img {
  max-width: 100%;
  height: auto;
}

4. Media Queries

@media (max-width: 768px) {
  body {
    background: black;
  }
}

Complete Responsive Example

<!DOCTYPE html>
<html>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
.container {
  width: 80%;
  margin: auto;
}

img {
  max-width: 100%;
  height: auto;
}

@media (max-width: 768px) {
  body {
    background: lightgray;
  }
}
</style>
</head>

<body>

<div class="container">
  <h1>Responsive Page</h1>
  <p>Resize the browser window to see the effect.</p>
  <img src="https://via.placeholder.com/600x300">
</div>

</body>
</html>

Output

Summary

  • HTML Entities display reserved characters safely.
  • They start with & and end with ;
  • Responsive Design adapts layout to screen sizes.
  • Use viewport, percentages, flexible images, and media queries.
