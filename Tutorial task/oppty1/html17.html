17. CSS & JS Integration

Modern web development is built on three core technologies: HTML (structure), CSS (presentation), and JavaScript (behavior). Understanding how to properly integrate CSS and JS into HTML is essential for building interactive and visually appealing websites.

Integrating CSS with HTML

There are three primary ways to add CSS to an HTML document. Each method has specific use cases depending on project size and maintainability.

1. External Stylesheet (Recommended for Real Projects)

This is the most professional and scalable approach. CSS is written in a separate file and linked using the <link> tag. 

<head>
  <link rel="stylesheet" href="styles.css">
</head>

Benefits:

  • Better maintainability
  • Reusable styles across pages
  • Faster page loading (cached CSS)

2. Internal Stylesheet

CSS rules are written inside the <style> element within the <head>. Useful for single-page styling. 

<head>
<style>
body { background-color: lightblue; }
h1 { color: navy; margin-left: 20px; }
</style>
</head>

3. Inline Styles

Styles are applied directly to HTML elements using the style attribute. This method is not recommended for large projects. 

<h1 style="color: blue; font-size: 24px;">Hello World</h1>
<p style="color: red;">This is a paragraph.</p>

Integrating JavaScript with HTML

JavaScript adds interactivity to web pages. It can be included either internally or externally.

1. External JavaScript

<body>
  <script src="script.js"></script>
</body>

Advantages:

  • Cleaner HTML structure
  • Reusable code
  • Better performance

2. Internal JavaScript

<script>
function myFunction() {
  document.getElementById("demo").innerHTML = "Hello JavaScript!";
}
</script>

Async and Defer

Control how scripts load:

  • async – Downloads script asynchronously.
  • defer – Executes script after HTML parsing completes.
<script src="script.js" async></script>
<script src="script.js" defer></script>

Complete Integration Example

<!DOCTYPE html>
<html>
<head>
<style>
.highlight { background-color: yellow; }
.hidden { display: none; }
</style>
</head>

<body>

<h1>CSS & JS Integration</h1>

<p id="demo">This paragraph will change.</p>

<button onclick="changeText()">Change Text</button>
<button onclick="toggleHighlight()">Toggle Highlight</button>

<script>
function changeText() {
  document.getElementById("demo").innerHTML = "Text changed!";
}

function toggleHighlight() {
  document.getElementById("demo").classList.toggle("highlight");
}
</script>

</body>
</html>

Output

Why Integration Matters

  • Separation of concerns (HTML structure, CSS style, JS behavior)
  • Improved maintainability
  • Better performance optimization
  • Scalable project architecture
