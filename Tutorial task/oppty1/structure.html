4. Structure of a Webpage

Understanding the basic structure of an HTML document.

HTML Document Structure

Every HTML document follows a standard structure that includes several essential elements: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <!-- Page content goes here -->
</body>
</html>

Breaking Down the Structure:

  1. The <!DOCTYPE html> declaration defines that this document is an HTML5 document.
  2. The <html> element is the root element of an HTML page.
  3. The <head> element contains meta information about the HTML page.
  4. The <title> element specifies a title for the HTML page (shown in the browser tab).
  5. The <body> element defines the document's body and contains all visible contents.
  6. The <h1> element defines a large heading.
  7. The <p> element defines a paragraph.

The <head> section contains metadata, while the <body> section contains visible content.

Core Structural Elements in Detail

1. Document Type Declaration

The DOCTYPE declaration tells the browser which version of HTML the page is using. It ensures the browser renders the page in standards mode. 

<!DOCTYPE html>

2. HTML Root Element

The <html> element wraps the entire document. The lang attribute specifies the language of the webpage. 

<html lang="en">
</html>

3. Head Section

The <head> section contains metadata such as character encoding, viewport configuration, page title, stylesheets, and scripts. 

<head>
    <meta charset="UTF-8">
    <title>My Page</title>
</head>

4. Body Section

The <body> contains all the visible content displayed in the browser window. This includes headings, paragraphs, images, links, and more. 

<body>
    <h1>Welcome</h1>
    <p>This is a paragraph.</p>
</body>

5. Semantic Structure Elements

Modern HTML uses semantic tags to clearly describe different parts of the webpage layout. 

<header>Header Content</header>
<nav>Navigation Links</nav>
<main>Main Content</main>
<section>Section Content</section>
<footer>Footer Content</footer>

6. DOM Tree Representation

HTML elements are organized in a tree-like hierarchy called the Document Object Model (DOM). Each element can have parent and child elements. 

html
 ├── head
 │    ├── meta
 │    └── title
 └── body
      ├── header
      ├── main
      └── footer

Complete Example

Here's a complete HTML document with all essential elements:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Web Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is the main content of our website.</p>
</body>
</html>
