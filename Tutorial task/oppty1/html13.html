13.HTML Semantic Elements

Semantic elements clearly describe their meaning to both browser and developer. They improve SEO and accessibility.

HTML Semantic Elements are tags that clearly describe their meaning to both the browser and the developer. Unlike non-semantic tags like <div> and <span>, semantic tags explain the purpose of the content inside them.

Why Use Semantic Elements?

1. They improve website structure and readability.
2. They help search engines understand your content better (SEO).
3. They improve accessibility for screen readers.
4. They make code more meaningful and easier to maintain.
5. They clearly define different parts of a webpage like header, footer, navigation, and sections.

<header> Tag

The header element represents introductory content like logo, title, navigation links. 

<header>
  <h1>My Website</h1>
  <p>Welcome to my site</p>
</header>

<nav> Tag

Defines navigation links. 

<nav>
  <a href="#">Home</a> |
  <a href="#">About</a> |
  <a href="#">Contact</a>
</nav>

<section> Tag

Defines a section in a document. 

<section style="background:#f2f2f2; padding:15px;">
  <h2>Services</h2>
  <p>We provide web development services.</p>
</section>

<article> Tag

Represents independent content like blog post. 

<article>
  <h2>Blog Title</h2>
  <p>This is a blog article.</p>
</article>

<aside> Tag

Represents sidebar content. 

<aside>
  <p>This is sidebar content.</p>
</aside>

<footer> Tag

Represents footer section. 

<footer>
  <p>© 2026 My Website</p>
</footer>

<main> Tag

The main element represents the main content of the document. There should be only one <main> element in a webpage. 

<main>
  <h2>Main Content Area</h2>
  <p>This is the primary content of the webpage.</p>
</main>

<figure> and <figcaption> Tags

The figure element is used to group media content like images, and figcaption provides a caption for that content. 

<figure>
  <img src="https://via.placeholder.com/150" alt="Sample Image">
  <figcaption>This is an example image.</figcaption>
</figure>

<mark> Tag

The mark element highlights important text. 

<p>This is <mark>important</mark> highlighted text.</p>

<time> Tag

The time element represents a specific date or time. 

<p>Meeting on <time datetime="2026-02-27">February 27, 2026</time></p>

<details> and <summary> Tags

The details element creates a collapsible section, and summary defines the visible heading. 

<details>
  <summary>Click to view more</summary>
  <p>This content is hidden until clicked.</p>
</details>
