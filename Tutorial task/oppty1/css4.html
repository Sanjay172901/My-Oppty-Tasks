4 CSS Colors

CSS Colors

Colors in CSS are used to give style to HTML elements. CSS supports several ways to specify colors including named colors, hexadecimal, RGB, RGBA, HSL, and HSLA values.

Color Names

CSS supports 140 named colors. These are simple to use but offer limited options: 

.red { color: red; }.blue { color: blue; }.green { color: green; }..purple { color: purple; }

Hexadecimal Colors

Hexadecimal colors are specified with a # followed by 3 or 6 hexadecimal characters (0-9, A-F): 

/* 6-digit hex */ .background { background-color: #ff6b6b; 
    /* Red */ } .text { color: #333333; /* Dark gray */ } 
.link { color: #4285f4; /* Blue */ }
 /* 3-digit hex (shorthand) */ .border { border-color: #f00; /* Red (same as #ff0000) */ } .
 shadow { box-shadow: 0 2px 4px #00000040; /* Black with alpha */ }

RGB Colors

RGB colors are specified with the rgb() function, which takes three parameters (red, green, and blue) each ranging from 0 to 255: 

.red { color: rgb(255, 0, 0); }
.green { background-color: rgb(0, 255, 0); }
.blue { border-color: rgb(0, 0, 255); }
.gray { color: rgb(128, 128, 128); }

RGBA Colors

RGBA colors are specified with the rgba() function, which takes four parameters (red, green, blue, and alpha) each ranging from 0 to 255 and 0 to 1 respectively: 

.semi-transparent {
    background-color: rgba(255, 0, 0, 0.5);
}

.overlay {
    background-color: rgba(0, 0, 0, 0.7);
}

.highlight {
    color: rgba(255, 255, 0, 0.8);
}

HSL Colors

HSL stands for Hue, Saturation, and Lightness. Hue is a degree on the color wheel (0-360), saturation is a percentage (0-100%), and lightness is a percentage (0-100%): 

.red { color: hsl(0, 100%, 50%); }            /* Red */
.green { background-color: hsl(120, 100%, 50%); }  /* Green */
.blue { border-color: hsl(240, 100%, 50%); }  /* Blue */
.pastel { background-color: hsl(300, 50%, 80%); }  /* Pastel purple */

HSLA Colors

HSLA is like HSL but with an additional alpha parameter (0-1) for transparency: 

.semi-transparent {
    background-color: hsla(120, 100%, 50%, 0.5); /* 50% transparent green */
}

.overlay {
    background-color: hsla(0, 0%, 0%, 0.7); /* 70% transparent black */
}

Color Keywords

CSS also supports special keywords for transparency:

.transparent {
    color: transparent;
}

.current {
    color: currentColor; /* Inherits the current text color */
}

Try It Yourself

Experiment with different color formats:

.example1 { color: red; }
.example2 { color: #3498db; }
.example3 { color: rgb(255, 165, 0); }
.example4 { color: hsl(200, 100%, 50%); }
