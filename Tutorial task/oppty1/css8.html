8. Padding

Controlling space inside elements

CSS Padding Properties

The CSS padding properties are used to generate space around an element's content, inside of any defined borders. Padding can be set for all sides of an element or for specific sides.

Padding Values

Padding values can be specified in several ways:

/* Length values */
.length-padding { padding: 20px; }

/* Percentage values */
.percentage-padding { padding: 5%; }

/* Initial and inherit */
.initial-padding { padding: initial; }
.inherit-padding { padding: inherit; }

Individual Padding Sides

You can set padding for specific sides of an element:

.top-padding { padding-top: 20px; }
.right-padding { padding-right: 15px; }
.bottom-padding { padding-bottom: 30px; }
.left-padding { padding-left: 10px; }

Padding Shorthand

The padding property is a shorthand for setting all padding properties in one declaration: 

/* All sides same */
.all-sides { padding: 20px; }

/* Vertical and horizontal */
.vertical-horizontal { padding: 10px 20px; }

/* Top, horizontal, bottom */
.top-horizontal-bottom { padding: 10px 20px 15px; }

/* Top Right Bottom Left */
.all-four { padding: 10px 20px 15px 25px; }

Padding vs Margin

Understanding the difference between padding and margin:

/* Padding – space inside the border */
.with-padding {
  padding: 20px;
  border: 2px solid #ff6b6b;
  background-color: #ffeeee;
}

/* Margin – space outside the border */
.with-margin {
  margin: 20px;
  border: 2px solid #667eea;
  background-color: #eeeeff;
}

Padding with Different Units

Padding can be specified using different CSS units:

.pixel-padding { padding: 20px; }

.em-padding { padding: 2em; } /* Relative to font size */

.rem-padding { padding: 1.5rem; } /* Relative to root font size */

.percent-padding { padding: 5%; } /* Relative to parent width */

.viewport-padding { padding: 2vw; } /* Relative to viewport width */

Percentage Padding

Percentage padding is always calculated relative to the width of the containing block: 

.percent-padding {
  padding: 10%; /* 10% of parent's width */
  width: 200px;
  background-color: #ff6b6b;
}

Negative Padding

Unlike margins, padding cannot be negative. Negative padding values are invalid: 

/* This will not work */
.invalid-padding {
  padding: -20px; /* Invalid - ignored */
}
