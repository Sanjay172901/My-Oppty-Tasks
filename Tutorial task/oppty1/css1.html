1. Introduction to CSS

CSS stands for Cascading Style Sheets. It is used to style HTML elements and control layout.

There are three ways to apply CSS: Inline, Internal, and External stylesheets. 


body {
    background-color: black;
    color: white;
}

h1 {
    font-size: 40px;
}

CSS makes your website visually attractive and responsive.

How CSS Works

CSS works by associating rules with HTML elements. A CSS rule contains:

Selector – Points to the HTML element
Declaration – Style applied
Property – What to change
Value – Setting assigned 


h1 {
    color: blue;
    font-size: 36px;
}

Ways to Add CSS

1. External CSS

Link an external file inside the head section:


<head>
<link rel="stylesheet" href="styles.css">
</head>

2. Internal CSS

Define CSS rules within the HTML document itself:


<style>
body {
    background-color: lightblue;
}
h1 {
    color: navy;
}
</style>

3. Inline CSS

Apply CSS directly to HTML elements using the style attribute:


<h1 style="color: blue;">Hello World</h1>
<p style="color: red;">This is a paragraph.</p>

CSS Comments

Comments in CSS are used to explain code and are ignored by the browser.

<!DOCTYPE html>

<body>

<h1>Welcome to CSS</h1>

</html>
