15. HTML Meta Tags

Meta tags provide information about the webpage to browsers and search engines. They are placed inside the <head> section of an HTML document.

<meta charset>

Defines character encoding. UTF-8 supports most languages and symbols. 

<meta charset="UTF-8">

<meta name="viewport">

Makes website responsive on mobile devices. 

<meta name="viewport" 
      content="width=device-width, initial-scale=1.0">

<meta name="description">

Provides page description for search engines (SEO). 

<meta name="description" 
      content="Learn HTML meta tags with examples.">

<meta name="keywords">

Defines keywords related to the page content. 

<meta name="keywords" 
      content="HTML, meta tags, SEO, tutorial">

<meta name="author">

Specifies the author of the webpage. 

<meta name="author" content="Your Name">

<meta http-equiv="refresh">

Automatically refreshes or redirects the page. 

<meta http-equiv="refresh" content="5">

<meta name="robots">

Controls search engine indexing behavior. 

<meta name="robots" content="index, follow">

<meta name="theme-color">

Changes browser address bar color on mobile devices. 

<meta name="theme-color" content="#243b55">

Open Graph (Facebook)

<meta property="og:title" content="Meta Tags Tutorial">
<meta property="og:description" content="Learn HTML Meta Tags">

Twitter Card

<meta name="twitter:card" content="summary">
<meta name="twitter:title" content="Meta Tags Tutorial">
