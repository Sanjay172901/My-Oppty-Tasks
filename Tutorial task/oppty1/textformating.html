6. Text Formatting

HTML provides several elements to format text and make it visually appealing or semantically meaningful. Some elements only change appearance, while others convey importance and meaning to search engines and assistive technologies.

Basic Text Formatting

Bold Text

Use <b> for bold text without semantic importance. Use <strong> when the text has strong importance or emphasis. 

<p>This is <b>bold text</b>.</p>
<p>This is <strong>strongly emphasized text</strong>.</p>

Output

Italic Text

Use <i> for stylistic italic text. Use <em> to emphasize text semantically. 

<p>This is <i>italic text</i>.</p>
<p>This is <em>emphasized text</em>.</p>

Output

Underlined and Marked Text

<u> underlines text. <mark> highlights text to draw attention. 

<p>This is <u>underlined text</u>.</p>
<p>This is <mark>highlighted text</mark>.</p>

Output

Other Text Formatting Elements

Small Text

Use <small> for side-comments and disclaimers. 

<p>This is normal text.</p>
<p><small>This is smaller text.</small></p>

Deleted and Inserted Text

<del> shows deleted content. <ins> shows newly inserted content. 

<p>My favorite color is <del>blue</del> <ins>red</ins>!</p>

Subscript and Superscript

<sub> is used for chemical formulas. <sup> is used for mathematical expressions. 

<p>Water is H<sub>2</sub>O.</p>
<p>The 2<sup>nd</sup> of October.</p>

Quotations

Use <q> for short inline quotations. Use <blockquote> for longer quotations. 

<p>Einstein said: <q>Imagination is more important than knowledge.</q></p>

<blockquote>
<p>The only way to do great work is to love what you do.</p>
<cite>- Steve Jobs</cite>
</blockquote>

Abbreviations

Use <abbr> to define abbreviations and provide full form on hover. 

<p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.</p>

Complete Example

<h1>My Article</h1>

<p>In this article, I will discuss <strong>important</strong> concepts in web development.</p>

<p>The <em>most crucial</em> aspect is understanding the basics.</p>

<p>Some people still use <del>outdated</del> <ins>modern</ins> techniques.</p>

<p>Water is H<sub>2</sub>O.</p>

<p>The area of a circle is πr<sup>2</sup>.</p>

<p>Einstein said: <q>Everything should be made as simple as possible.</q></p>

<p><small>Note: This article is for educational purposes.</small></p>

Output
