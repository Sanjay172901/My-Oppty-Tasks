11. HTML Tables

HTML tables allow web developers to arrange data into rows and columns.

Basic Table Example

<table>
   <tr>
      <th>Name</th>
      <th>Age</th>
   </tr>
   <tr>
      <td>John</td>
      <td>25</td>
   </tr>
</table>

Table Elements

<table> – Defines the table container.

<tr> – Defines a table row.

<th> – Defines a header cell (bold & centered).

<td> – Defines a data cell.

Table Structure Tags

<thead> – Groups header content.

<tbody> – Groups body content.

<tfoot> – Groups footer content.

Table Attributes

colspan – Spans across multiple columns.

<th colspan="2">Contact Info</th>

rowspan – Spans across multiple rows.

<td rowspan="2">John Doe</td>

Complete Table Structure Example

A full table normally contains <table>, <thead>, <tbody>, and <tfoot>. This makes the table more organized and readable. 

<table>
   <thead>
      <tr>
         <th>Product</th>
         <th>Price</th>
      </tr>
   </thead>

   <tbody>
      <tr>
         <td>Laptop</td>
         <td>$800</td>
      </tr>
   </tbody>

   <tfoot>
      <tr>
         <td>Total</td>
         <td>$800</td>
      </tr>
   </tfoot>
</table>

More Table Attributes

border (Old Method)

Adds border around table. Modern HTML uses CSS instead. 

<table border="1">

width

Defines table width. 

<table width="100%">

cellpadding (Old Method)

Adds space inside table cells. 

<table cellpadding="10">

cellspacing (Old Method)

Adds space between table cells. 

<table cellspacing="5">

colspan Example

<table border="1">
   <tr>
      <th colspan="2">Full Name</th>
   </tr>
   <tr>
      <td>John</td>
      <td>Doe</td>
   </tr>
</table>

rowspan Example

<table border="1">
   <tr>
      <th>Name</th>
      <th>Email</th>
   </tr>

   <tr>
      <td rowspan="2">John Doe</td>
      <td>john@email.com</td>
   </tr>

   <tr>
      <td>john.work@email.com</td>
   </tr>
</table>

Styling Tables with CSS

Instead of using old HTML attributes like border, modern websites use CSS. 

<style>
table {
   border-collapse: collapse;
   width: 100%;
}

th, td {
   border: 1px solid black;
   padding: 8px;
   text-align: left;
}

th {
   background-color: #f2f2f2;
}
</style>

• border-collapse: Removes double borders
• padding: Adds spacing inside cells
• background-color: Changes header color

Try It Yourself

<!DOCTYPE html>
<html>
<body>

<h1>Employee Information</h1>

<table style="border-collapse: collapse; width:100%;" border="1">
<thead>
<tr>
<th>ID</th>
<th>Name</th>
<th>Department</th>
<th>Salary</th>
</tr>
</thead>

<tbody>
<tr>
<td>001</td>
<td>John Smith</td>
<td>Engineering</td>
<td>$75,000</td>
</tr>

<tr>
<td>002</td>
<td>Jane Doe</td>
<td>Marketing</td>
<td>$65,000</td>
</tr>
</tbody>

<tfoot>
<tr>
<td colspan="3">Total Employees</td>
<td>2</td>
</tr>
</tfoot>

</table>

</body>
</html>
