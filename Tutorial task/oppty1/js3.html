3. Variables

Learn about variables and data types in JavaScript

What are Variables?

Variables are containers for storing data values. In JavaScript, you can declare variables using var, let, or const. Think of a variable as a labeled box where you can put information.

Variable Declaration

There are three ways to declare variables in JavaScript. The modern industry standard is to use let and const.

// Using var (old way, function-scoped)
var firstName = "John"; 

// Using let (modern way, block-scoped)
let lastName = "Doe"; 

// Using const (for constants, block-scoped)
const birthYear = 1990;

Var vs Let vs Const

Understanding the difference between these keywords is vital for writing bug-free code:

var

  • Function-scoped
  • Can be redeclared
  • Hoisted to top

let

  • Block-scoped
  • Cannot redeclare
  • Supports reassignment

const

  • Block-scoped
  • Non-reassignable
  • Must be initialized

Data Types

JavaScript provides several primitive and complex data types to handle information:

let name = "John";           // String
let age = 30;               // Number
let isStudent = true;       // Boolean
let x;                      // Undefined
let y = null;               // Null
let sym = Symbol("id");     // Symbol (Unique)
let big = 9007199254740991n;// BigInt (Large numbers)

// Complex Types
let person = {name: "John"}; // Object
let colors = ["red", "blue"];// Array (Type of Object)

Type Checking

You can use the typeof operator to find out the data type of a JavaScript variable at any time during execution.

let message = "Hello";
console.log(typeof message); // "string"

let score = 95;
console.log(typeof score);   // "number"
