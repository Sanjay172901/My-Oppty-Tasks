7. Links & Anchors

Links are created using the <a> (anchor) tag. They allow users to navigate between pages, websites, or sections within a page.

Basic Link Syntax

The href attribute specifies the destination URL. 

<a href="https://www.google.com">Visit Google</a>

Output

Open Link in New Tab

Use target="_blank" to open a link in a new browser tab. 

<a href="https://www.google.com" target="_blank">Open Google</a>

Output

Link to Another Page (Relative Link)

You can link to another file in your project using a relative path. 

<a href="about.html">Go to About Page</a>

Output

Anchor Link (Jump to Section)

Anchor links allow navigation within the same page using the id attribute. 

<a href="#contact">Go to Contact Section</a>

<h2 id="contact">Contact Section</h2>
<p>This is the contact section.</p>

Output

Email Link

You can create a clickable email link using mailto: 

<a href="mailto:example@gmail.com">Send Email</a>

Output

Why Links Are Important

  • Enable navigation between pages
  • Connect internal and external resources
  • Improve user experience
  • Essential for website structure
