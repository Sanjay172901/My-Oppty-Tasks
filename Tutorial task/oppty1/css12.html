12 Font

CSS Font Properties

CSS provides several properties to control the appearance of fonts, including font family, size, weight, style, and variant. These properties help create visually appealing and readable text.

Font Family

The font-family property specifies the font for an element. You can specify multiple fonts as a fallback system: 

.serif-font { font-family: "Times New Roman", Times, serif; } .sans-serif-font { font-family: Arial, Helvetica, sans-serif; } 
    .monospace-font { font-family: "Courier New", Courier, monospace; } .generic-font { font-family: sans-serif; }

Font Size

The font-size property sets the size of the font: 

/.pixel-size { font-size: 16px; } .em-size { font-size: 1.2em; 
    /* Relative to parent element */ } .rem-size { font-size: 1.5rem; 
        /* Relative to root element */ } .percent-size { font-size: 120%;
             /* Percentage of parent element */ } .keyword-size { font-size: large; } 
             .viewport-size { font-size: 4vw; /* Relative to viewport width */ }

Font Weight

The font-weight property sets how thick or thin characters in text should be displayed 

.normal-weight { font-weight: normal; /* 400 */ }
 .bold-weight { font-weight: bold; /* 700 */ }
 .light-weight { font-weight: 300; } .medium-weight { font-weight: 500; } 
 .extra-bold { font-weight: 800; } .numeric-weight { font-weight: 600; }

Font Style

The font-style property specifies the style of the font: 

.normal-style { font-style: normal; } 
.italic-style { font-style: italic; } 
.oblique-style { font-style: oblique; }

Font Variant

The font-variant property specifies whether text should be displayed in small-caps 

.normal-variant { font-variant: normal; } .small-caps { font-variant: small-caps; }

Font Shorthand

The font property is a shorthand for setting all font properties in one declaration: 

/* font: font-style font-variant font-weight font-size/line-height font-family 
*/ .shorthand-font { font: italic small-caps bold 16px/1.5 Arial, sans-serif; } 
.simple-shorthand { font: 14px Georgia, serif; } .with-line-height 
{ font: 18px/1.6 "Helvetica Neue", Helvetica, Arial, sans-serif; }

Web Fonts

Using custom fonts with @font-face:

@font-face { font-family: "CustomFont"; src: url("custom-font.woff2") format("woff2"),
 url("custom-font.woff") format("woff"); font-weight: normal; font-style: normal; } 
 .custom-font { font-family: "CustomFont", Arial, sans-serif; }

Google Fonts

Including Google Fonts in your project:

./* In HTML head */  
/* In CSS */ .roboto-font { font-family: 'Roboto', sans-serif; }
