11. DOM Manipulation

Interact with HTML and CSS to build dynamic, responsive web experiences

What is the DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can connect to the page.

Selecting Elements

Before you can change an element, you must find it. JavaScript provides several methods to select elements:

// Find element by ID
const element = document.getElementById("intro");

// Find elements by tag name
const paragraphs = document.getElementsByTagName("p");

// Find elements by class name
const items = document.getElementsByClassName("list-item");

// Find elements by CSS selectors (Modern & Preferred)
const header = document.querySelector(".main-header");
const allButtons = document.querySelectorAll("button");

Changing HTML Content

The easiest way to modify the content of an HTML element is by using the innerHTML or textContent property.

document.getElementById("myH1").innerHTML = "New Heading";
document.getElementById("myP").textContent = "New Paragraph Text";

innerHTML: Can parse HTML tags (e.g., <b>Bold</b>).
textContent: Handles only raw text, making it safer for user-generated content.

Changing CSS Styles

You can change the style of an HTML element by accessing the style property.

const box = document.getElementById("box");

box.style.color = "blue";
box.style.backgroundColor = "yellow";
box.style.display = "none"; // Hide element

Attributes and Classes

You can also manipulate attributes and class lists directly.

// Attributes
element.setAttribute("src", "image.jpg");

// Class List (Best practice for styling)
element.classList.add("active");
element.classList.remove("hidden");
element.classList.toggle("dark-mode");
