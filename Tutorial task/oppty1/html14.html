14. HTML Multimedia Elements

Multimedia elements allow you to add images, audio, video, and other external content into your webpage.

<img> Tag

Used to display images on a webpage. 

<img src="https://via.placeholder.com/200" alt="Sample Image">

<audio> Tag

Used to embed sound content. 

<audio controls>
  <source src="https://www.w3schools.com/html/horse.mp3" type="audio/mpeg">
  Your browser does not support audio.
</audio>

<video> Tag

Used to embed video content. 

<video width="250" controls>
  <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4">
  Your browser does not support video.
</video>

<picture> Tag

Allows responsive images depending on screen size. 

<picture>
  <source media="(min-width: 650px)" 
          srcset="https://via.placeholder.com/400">
  <img src="https://via.placeholder.com/200" 
       alt="Responsive Image">
</picture>

<iframe> Tag

Used to embed another webpage inside your page. 

<iframe src="https://example.com" 
        width="300" height="150">
</iframe>

<embed> Tag

Used to embed external content like PDF. 

<embed src="https://www.w3.org/WAI/ER/tests/xhtml/testfiles/resources/pdf/dummy.pdf" 
       width="300" height="150">

<object> Tag

Used to embed media like images, PDF, or HTML. 

<object data="https://via.placeholder.com/200" 
        width="200" height="150">
</object>

<source> Tag

The source tag defines multiple media resources for audio or video. The browser chooses the first supported format. 

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Your browser does not support audio.
</audio>

<track> Tag

The track tag is used inside the video or audio element to add subtitles or captions. It improves accessibility and supports multiple languages. 

<video width="250" controls>
  <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4">
  <track src="https://www.w3schools.com/tags/movie_en.vtt"
         kind="subtitles"
         srclang="en"
         label="English">
</video>

<map> and <area> Tags

Used to create clickable areas on an image (image map). 

<img src="https://via.placeholder.com/300"
     usemap="#image-map">

<map name="image-map">
  <area shape="rect"
        coords="0,0,150,150"
        href="https://example.com"
        alt="Clickable Area">
</map>

<figure> with Multimedia

The figure tag groups media content with a caption using figcaption. 

<figure>
  <video width="200" controls>
    <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4">
  </video>
  <figcaption>Sample Video Example</figcaption>
</figure>

<iframe> (YouTube Example)

Iframe can embed YouTube videos and external platforms. 

<iframe width="300" height="170"
src="https://www.youtube.com/embed/tgbNymZ7vqY"
allowfullscreen>
</iframe>

<canvas> Tag

The canvas element is used to draw graphics using JavaScript. 

<canvas id="myCanvas" width="200" height="100"></canvas>

<script>
var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.fillStyle = "blue";
ctx.fillRect(20, 20, 150, 50);
</script>

<svg> Tag

SVG is used to create scalable vector graphics. 

<svg width="200" height="100">
  <circle cx="50" cy="50" r="40"
          stroke="black"
          stroke-width="3"
          fill="red" />
</svg>
