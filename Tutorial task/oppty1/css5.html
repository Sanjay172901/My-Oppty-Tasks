5. CSS Border

CSS Border Properties

The CSS border properties allow you to specify the style, width, and color of an element's border. There are several border properties that can be used individually or as shorthand properties. For example, you can use the border property to set all the border properties at once, or you can use the border-width, border-style, and border-color properties to set each property individually.

Border Color

The border-color property sets the color of an element's border.

.none { border-style: none; /* No border */ } 
.dotted { border-style: dotted; /* Dotted border */ }
.solid { border-style: solid; /* Solid border */ } 
.double { border-style: double; /* Double border */ } 
.groove { border-style: groove;  /* 3D grooved border */ }
.outset { border-style: outset; /* 3D outset border */ }

Border Width

The border-width property sets the width of an element's border.

 thin { border-style: solid; border-width: thin; } 
 .medium { border-style: solid; border-width: medium; } 
 .thick { border-style: solid; border-width: thick; }
 .specific { border-style: solid; border-width: 5px; }
  .multiple-widths { border-style: solid; border-width: 2px 10px 4px 20px; 
    /* top right bottom left */ }

Border Color

The border-color property sets the color of an element's border.

.red-border { border-style: solid; border-color: red; } 
.hex-border { border-style: solid; border-color: #ff6b6b; } 
.rgb-border { border-style: solid; border-color: rgb(255, 107, 107); }
 .transparent-border { border-style: solid; border-color: transparent; } 
 .multiple-colors { border-style: solid; border-color: red green blue yellow; 
    /* top right bottom left */ }

Individual Border Sides

The border-color property sets the color of an element's border.

.top-border { border-top-style: dotted; border-top-width: 2px;
     border-top-color: red; } 
.right-border { border-right-style: dashed; border-right-width: 5px;
     border-right-color: blue; } 
.bottom-border { border-bottom-style: solid; border-bottom-width: 3px;
     border-bottom-color: green; } 
.left-border { border-left-style: double; border-left-width: 4px;
     border-left-color: purple; }

Border Shorthand Properties

The border property is a shorthand property for the following individual border properties: 

.* Border shorthand for all sides */ 
.shorthand-all { border: 2px solid #ff6b6b; } 
/* Individual side shorthand */ 
.border-top { border-top: 3px dotted blue; } 
.border-right { border-right: 5px dashed green; } 
.border-bottom { border-bottom: 2px double purple; } 
.border-left { border-left: 4px groove orange; }

Border Radius

The border-radius property is used to add rounded corners to an element. It can take pixel values, percentage values, or global values like inherit or initial. 

.rounded-corners { border: 2px solid #ff6b6b; border-radius: 10px; } 
.circle { border: 2px solid #ff6b6b; border-radius: 50%; 
    /* Creates a circle */ width: 100px; height: 100px; } 
.elliptical { border: 2px solid #ff6b6b; border-radius: 50px/25px; 
    /* Elliptical corners */ width: 200px; height: 100px; } 
.individual-corners { border: 2px solid #ff6b6b; border-radius: 10px 20px 30px 40px;
     /* top-left top-right bottom-right bottom-left */ } 
.specific-corners { border: 2px solid #ff6b6b;
     border-top-left-radius: 20px; border-top-right-radius: 40px; 
     border-bottom-right-radius: 20px; border-bottom-left-radius: 10px; }
