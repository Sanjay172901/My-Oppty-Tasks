10. HTML Lists

HTML provides three types of lists: ordered lists, unordered lists, and definition lists.

Unordered List

The <ul> tag creates a bulleted list. Each item inside the list must be written using the <li> tag. 

<ul>
   <li>Apple</li>
   <li>Banana</li>
   <li>Orange</li>
</ul>

Ordered List

The <ol> tag creates a numbered list. Items are also written using <li>. 

<ol>
   <li>First item</li>
   <li>Second item</li>
   <li>Third item</li>
</ol>

Definition List

The <dl> tag creates a description list. It is used for terms and definitions. 

<dl>
   <dt>HTML</dt>
   <dd>HyperText Markup Language</dd>
   <dt>CSS</dt>
   <dd>Cascading Style Sheets</dd>
</dl>

List Attributes (Ordered List)

The <ol> tag creates a numbered list. Items are also written using <li>.

type Attribute

Specifies numbering type: 

<ol type="1">   Default Numbers</ol>
<ol type="A">   Uppercase Letters</ol>
<ol type="a">   Lowercase Letters</ol>
<ol type="I">   Uppercase Roman</ol>
<ol type="i">   Lowercase Roman</ol>

start Attribute

Defines the starting number of an ordered list. 

<ol start="5">
   <li>Item 5</li>
   <li>Item 6</li>
</ol>

reversed Attribute

Displays list in descending order. 

<ol reversed>
   <li>Item 3</li>
   <li>Item 2</li>
</ol>

Styling Lists with CSS

You can change bullet style using the list-style-type property. 

<ul style="list-style-type: square;">
   <li>Square Bullet</li>
</ul>

<ul style="list-style-type: circle;">
   <li>Circle Bullet</li>
</ul>

<ul style="list-style-type: none;">
   <li>No Bullet</li>
</ul>

Nested Lists

<ul>
   <li>Fruits
       <ul>
           <li>Apple</li>
           <li>Banana</li>
       </ul>
   </li>
</ul>

    <li>Java</li>
    <li>C++</li>
</ol>

<h2>Favorite Web Technologies</h2>
<ul style="list-style-type: square;">
    <li>HTML5</li>
    <li>CSS3</li>
    <li>React</li>
    <li>Node.js</li>
</ul>

<h2>Web Development Concepts</h2>

    <dt>Responsive Design</dt>
    <dd>Design that adapts to different screen sizes</dd>

    <dd>Design usable by people with disabilities</dd>
</dl>

</body>
</html>
