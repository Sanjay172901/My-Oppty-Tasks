6. Background

CSS Background Properties

CSS background properties are used to define the background effects for elements.There are several background properties that can be used individually or as a shorthand property.

Background Color

The background-color property specifies the background color of an element: 

.background-example { background-color: #ff6b6b; } .transparent-bg { background-color: transparent; } 
.named-color { background-color: lightblue; }

Background Image

The background-image property specifies the background image of an element: 

background-image { background-image: url("image.jpg"); }

Background Repeat

The background-repeat property specifies how a background image should be repeated : 

    * Default - repeat both directions */ .repeat-both { background-image: url("pattern.png"); } 
     /* Repeat horizontally only */ 
    .repeat-x { background-image: url("pattern.png"); background-repeat: repeat-x; } 
    /* Repeat vertically only */ .repeat-y
     { background-image: url("pattern.png"); background-repeat: repeat-y; } 
     /* No repeat */ .no-repeat { background-image: 
     url("logo.png"); background-repeat: no-repeat; }

Background Position

The background-position property specifies the position of a background image: 

    * Position keywords */ .top-left { background-image: url("image.png");
     background-position: top left; } 
    /* Center */ .center { background-image: url("image.png");
     background-position: center; } 
    /* Bottom right */ .bottom-right { background-image: url("image.png"); 
    background-position: bottom right; } 
    /* Percentage values */ .percent-position { background-image: url("image.png");
     background-position: 25% 75%; } 
    /* Length values */ .length-position { background-image: url("image.png");
     background-position: 10px 20px; }

Background Attachment

The background-attachment property specifies whether a background image should scroll with the rest of the page or be fixed: 

.scroll { background-image: url("image.png"); background-attachment: scroll; /* Default */ } 
.fixed { background-image: url("image.png"); background-attachment: fixed; }

Background Size

The background-size property specifies the size of the background images: 

.auto-size { background-image: url("image.png"); background-size: auto; /* Default */ } 
.cover { background-image: url("image.png"); background-size: cover; /* Scale to cover entire element */ }
 .contain { background-image: url("image.png"); background-size: contain; /* Scale to fit inside element */ } 
 .specific-size { background-image: url("image.png"); background-size: 100px 80px; /* Specific dimensions */ }
  .percentage-size { background-image: url("image.png"); background-size: 50% 75%; /* Percentage of element */ }

Background Shorthand

The background property is a shorthand for setting all background properties in one declaration: 

.shorthand-example { background: #ff6b6b url("image.png") no-repeat center/cover; }
 /* The order is: background: color image repeat position/size attachment;
  */ .detailed-shorthand { background: linear-gradient(135deg, #ff6b6b, #ee5a24) url("pattern.png") repeat-x top left scroll; }

Multiple Backgrounds

The background property can be used to set multiple backgrounds for an element: 

multiple-backgrounds { background-image: url("image1.png"), url("image2.png"),
 linear-gradient(to bottom, #ff6b6b, #ee5a24); background-position: right bottom, left top, center;
  background-repeat: no-repeat, no-repeat, no-repeat; }
