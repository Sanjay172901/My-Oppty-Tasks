3. CSS Selectors

Learn how to target HTML elements with CSS.

What are CSS Selectors?

CSS selectors are patterns used to select HTML elements you want to style. They are the part of a CSS rule that actually selects the content.

Basic Selectors

Element Selector

Selects all elements of a specified type.

p { color: blue; }
h1 { font-size: 2em; }

ID Selector

Selects a single element with a specific id attribute (use # symbol).

#header { background-color: #333; color: white; padding: 20px; }

Class Selector

Selects all elements with a specific class attribute (use . symbol).

.highlight { background-color: yellow; }
.button { padding: 10px 20px; border: none; border-radius: 5px; }

Universal Selector

Selects all elements in the document (use * symbol).

* { margin: 0; padding: 0; box-sizing: border-box; }

Combinator Selectors

Descendant Selector

Selects elements that are descendants of another element.

div p { color: blue; }

Child Selector

Selects elements that are direct children (use > symbol).

div > p { color: red; }

Adjacent Sibling Selector

Selects an element directly after another (use + symbol).

h1 + p { font-weight: bold; }

General Sibling Selector

Selects all siblings after an element (use ~ symbol).

h1 ~ p { color: green; }

Attribute Selectors

[attribute]

Selects elements with a specific attribute.

[title] { border: 1px solid blue; }

[attribute="value"]

Selects elements with a specific attribute value.

[type="text"] { width: 200px; padding: 5px; }

[attribute~="value"]

Selects elements with an attribute containing a specified word.

[title~="flower"] { color: green; }

[attribute|="value"]

Selects elements with an attribute value that starts with specified value.

[lang|="en"] { color: blue; }

Pseudo-class Selectors

:hover

Selects elements when you hover over them.

a:hover { color: red; }

:focus

Selects elements when they have focus.

input:focus { border: 2px solid blue; }

:nth-child()

Selects elements based on position among siblings.

tr:nth-child(even) { background-color: #f2f2f2; }
tr:nth-child(odd) { background-color: #ffffff; }

Pseudo-element Selectors

::before

Inserts content before an element.

p::before { content: "→ "; color: blue; }

::after

Inserts content after an element.

p::after { content: " ←"; color: red; }
